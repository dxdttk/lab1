MenYou

CSE110 Project

MenYou is an iOS application created to enhance the restaurant dining experience.

Features

  1. MenYou allows for user login or guest use.
  2. When choosing to long in, MenYou can utilize the iPhone's fingerprint scanner.
  3. Not reinventing the wheel, MenYou uses Yelp's API in order to find nearby restaurants.
  4. MenYou's features come to light in its user's settings.
